# AYGI-VAIC

Repository tổng hợp các dự án Smart Traffic.

## Các dự án

### SmartTrafficSimulator

Mô phỏng bản sao số ngã tư giao thông với frontend Next.js/Three.js và backend FastAPI điều khiển tín hiệu realtime.

- Thư mục dự án: [`SmartTrafficSimulator/`](./SmartTrafficSimulator/)
- Hướng dẫn chạy: [`SmartTrafficSimulator/README.md`](./SmartTrafficSimulator/README.md)
- Backend: [`SmartTrafficSimulator/backend/`](./SmartTrafficSimulator/backend/)
- Frontend: [`SmartTrafficSimulator/frontend/`](./SmartTrafficSimulator/frontend/)

### Smart-Traffic-Monitoring-System

Hệ thống giám sát giao thông thực tế dùng YOLO/ONNX và ByteTrack để nhận diện, đếm phương tiện, ước lượng tốc độ; FastAPI và React cung cấp dashboard cùng AI assistant theo thời gian thực.

- Thư mục dự án: [`Smart-Traffic-Monitoring-System/`](./Smart-Traffic-Monitoring-System/)
- Hướng dẫn cài đặt và chạy: [`Smart-Traffic-Monitoring-System/README.md`](./Smart-Traffic-Monitoring-System/README.md)
- Backend: [`Smart-Traffic-Monitoring-System/backend/`](./Smart-Traffic-Monitoring-System/backend/)
- Frontend: [`Smart-Traffic-Monitoring-System/frontend/`](./Smart-Traffic-Monitoring-System/frontend/)

### traffic_red_light_jetson

Phần mềm điều khiển jetson nano giúp quản lí việc đưa giữ liệu lên trang web, xử lí AI tại biên và xử lí tín hiệu đèn giao thông
